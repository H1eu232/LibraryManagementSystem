# Library Management System

## Giới thiệu
Đây là hệ thống quản lý thư viện được xây dựng bằng ngôn ngữ **Python**. Dự án được thiết kế để giúp đơn giản hóa quy trình quản lý sách, theo dõi độc giả và xử lý các giao dịch mượn trả.

## Chức năng chính
Hệ thống bao gồm các mô-đun chức năng sau:

- **Quản lý sách (Book Management):** Thêm, sửa, xóa và tìm kiếm thông tin sách.
- **Quản lý thành viên (Member Management):** Quản lý hồ sơ độc giả, thẻ thư viện.
- **Quản lý mượn/trả (Borrow & Return):** Theo dõi phiếu mượn, ngày trả và tình trạng sách.
- **Quản lý phạt (Fine Management):** Tính toán phí phạt cho các trường hợp trả muộn hoặc làm hỏng sách.
- **Báo cáo (Reporting):** Xuất báo cáo thống kê hoạt động của thư viện.

## Yêu cầu hệ thống
- Python 3.x
- Các thư viện hỗ trợ (nếu có)
